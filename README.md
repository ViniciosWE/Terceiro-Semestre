## Terceiro Semestre - Análise e Desenvolvimento de Sistemas

No terceiro semestre, aprofundei meus conhecimentos em **Banco de Dados**, explorando SQL avançado, incluindo DDL, DML, funções de agregação, subconsultas, junções, visões, restrições e gatilhos.  

Estudei **Programação Orientada a Objetos em Java**, desenvolvendo habilidades em herança, abstração, tratamento de exceções, arquivos, interfaces gráficas e geração de relatórios.  

Em **Estruturas de Dados**, aprendi a implementar listas encadeadas, listas duplamente encadeadas, listas circulares, pilhas, filas, árvores e coleções, aplicando conceitos de orientação a objetos para organizar e manipular dados de forma eficiente.  

Além disso, avancei em **Engenharia de Software**, com foco em modelagem de sistemas orientada a objetos e diagramas UML, consolidando a capacidade de análise e projeto de software de forma estruturada e profissional.
